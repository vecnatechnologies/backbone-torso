Torso.Mixins.pollingMixin Class

Defined in: modules/mixins/pollingMixin.js:12
Module: Torso

Periodic Polling Object to be mixed into Backbone Collections and Models.

The polling functionality should only be used for collections and for models that are not part of any collections. It should not be used for a model that is a part of a collection.
